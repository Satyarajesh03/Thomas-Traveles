Thomas Travels – Driver Management System
Project Overview
Thomas Travels is a Java-based console application developed to manage driver details and travel-related operations for a travel agency. The project focuses on maintaining driver records, identifying vehicle categories, tracking traveled distance, and performing driver-related operations using Java Collections and Stream API.
The application demonstrates core Java concepts such as:
   •	Object-Oriented Programming (OOP) 
   •	Collections Framework 
   •	Java Stream API 
   •	Encapsulation 
   •	Constructors 
   •	Method Overriding 
   •	Filtering and Searching Data 
   •	Sorting Operations 
This project helps automate basic driver management tasks in a travel organization.
